# E-Commerce Sales Analysis

Advanced data analysis project simulating real-world business KPIs.

## Objectives
- Calculate total sales and profit
- Compute profit margins
- Analyze sales by region
- Identify most profitable category

## Technologies Used
Python
Pandas

## Key Business Insights
- Electronics generated highest revenue
- Furniture contributed highest profit
- North region showed strong performance
